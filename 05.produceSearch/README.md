## 画像のプロダクト検索

| プロパティ | 説明 |
| ----- | ----- |
| product_id | プロダクトの識別子。 |
| score | プロダクトの信頼度。 |
| description | プロダクトの説明。 |
| bounding_poly | プロダクトの境界ボックス。 |
| product_category | プロダクトのカテゴリ。 |
| product_display_name | プロダクトの表示名。 |
| product_labels | プロダクトのラベル。 |
